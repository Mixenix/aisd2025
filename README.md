# Алгоритмы и структуры данных

Лабораторные работы по курсу «Алгоритмы и структуры данных», НИЯУ МИФИ, 1 курс.

## Содержание

| Лабораторная | Тема |
|---|---|
| 1 | Основы работы с памятью, указатели, ввод-вывод |
| 2 | Очереди на списке и на векторе. Сравнение реализаций |
| 3a | Поиск и удаление подстроки |
| 3b | Хеш-таблицы |
| 4a | Бинарное дерево поиска. Операции вставки, поиска, удаления. Визуализация через Graphviz (DOT) |
| 4b | Деревья. Обходы. Замеры времени выполнения |
| 5 | Графы. BFS, поиск кратчайшего пути. Применение к задаче поиска выхода из лабиринта |

## Стек

- **C** (C11)
- **CMake** — сборка
- **Graphviz** — визуализация деревьев и графов
- **readline** — ввод с автодополнением

## Сборка

Каждая лабораторная собирается отдельно через CMake:

```bash
cd 4a && mkdir build && cd build && cmake .. && make
```
